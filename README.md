# odin-recipes

This is a small project to practice some of the foundational HTML concepts I've learned from https://www.theodinproject.com so far.

I'll continue updating this readme as I progress through future iterations using updated knowledge and techniques.

## Part 1

In the first part of this project, I used foundational HTML to set up the initial folder and page structure of my recipe website. I added individual pages for a few different recipes that I'm personally fond of, adding links to each in `index.html` and backlinks to `index.html` on each page.

I also added a directory to contain the images and added these images to their respective recipe pages.

I filled out each recipe page with descriptions, unordered lists of ingredients, and ordered lists of instructions, appropriately using headings to organize the structure of the page.

Though I was already very familiar with HTML by the time I worked on the first part of this project, I did have a few key takeaways:

- First, the usage of shortcuts in VSCode in order to streamline the process of writing code, such as creating cursors on multiple lines with `ctrl+alt+up/down arrows` to quickly create multiple list items. 

- Second, getting used to the workflow of git, making sure to commit each iteration of the project, and learning the importance of clear and concise commit messages.
